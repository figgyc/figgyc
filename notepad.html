Welcome to AutoNotepad! This is a completely local based app and no data is sent to any servers. Here's some tips:
  • Changes are saved automatically.
  • You can use common editor keyboard shortcuts such as Ctrl+Z or Ctrl+B.
  • You can drag in images from other browser tabs or your computer. Note that you can't resize images in Chrome and Edge but you can in other browsers such as Firefox.
  • Use the code button on the toolbar to embed YouTube videos.
